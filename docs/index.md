title: "blogmdsvex"
---

A small, fully **prerendered** blog built with SvelteKit, Svelte 5, MDsveX and
Tailwind. Blog posts are plain Markdown files kept alongside the project; at
build time every post becomes a static HTML page, so the published site needs no
server at runtime.

The catalogue below covers three areas: **reading the blog** (the home listing
and individual post pages), **authoring content** (how posts are written), and
the **site and build** setup (prerendering, layout, styling, and the development
file-access safeguard).
