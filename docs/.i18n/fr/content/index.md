order: 20

Comment les articles de blog sont écrits et où ils se trouvent. Les articles
sont des fichiers Markdown ordinaires conservés dans le dossier de contenu du
projet — pas de base de données, pas d'écran d'administration. Le nom de chaque
fichier devient l'adresse web de l'article, et un petit bloc de détails en haut
du fichier fournit son titre et sa description.
